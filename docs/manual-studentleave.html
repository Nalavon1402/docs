›คู่มือการใช้งาน

คู่มือการใช้งาน

  • การเข้าสู่ระบบ
  • การจัดตารางสอน
  • การเพิ่มข้อมูลนักเรียนเข้าใหม่
  • บันทึกการลงเวลาของนักเรียนแบบ Manual และ แบบกลุ่ม
  • การยืม-คืนบัตรสำรอง
  • การบ้าน
  • แจ้งข่าวสาร GPA
  • คู่มือการสร้างกิจกรรมโรงเรียน
  • บันทึกการลารายบุคคล และ การลารายกลุ่ม
  • บันทึกการตัดคะแนนของนักเรียน
  • บันทึกข้อมูลสุขภาพ
  • บันทึกการลงเวลาของบุคลากรแบบ Manual
  • การแจ้งข่าวสารแบบกลุ่ม แบบเดี่ยว แบบทั้งโรงเรียน

บันทึกการลารายบุคคล และ การลารายกลุ่ม

1. บันทึกการลารายบุคคล

  • ไปที่เมนูการลงเวลา
  • บันทึกการลา
  • เพิ่มการลารายบุคคล
  • กรอกชื่อนักเรียนที่แจ้งลา เลือกประเภทการลา วันเวลา และหมายเหตุการลา
  • กดปุ่มเพิ่ม

หมายเหตุ : สามารถลาย้อนหลังหรือลาล่วงหน้าได้ ในกรณีหมอนัดล่วงหน้า

2. บันทึกการลารายกลุ่ม

  • ไปที่เมนูการลงเวลา
  • บันทึกการลา
  • เพิ่มการลารายกลุ่ม
  • กรอกชื่อกลุ่ม เลือกประเภทการลา (เช่นนักเรียนชั้น ม.2/1 ลาไปทำกิจกรรม) วันเวลา และหมายเหตุการลา
  • กดปุ่มเพิ่ม

หมายเหตุ : สามารถลาย้อนหลัง หรือลาล่วงหน้าได้ กรณีในตารางหรือครูที่ปรึกษาแจ้งวันทำกิจกรรมล่วงหน้า
